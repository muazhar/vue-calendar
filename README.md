# vue-calendar
Time efficient implementation of Vue Calendar Control

## Install
Clone Repository, Change main.js with your firebase key
```sh
firebase.initializeApp({
  apiKey: "",
    authDomain: "",
    databaseURL: "",
    projectId: "",
    storageBucket: "",
    messagingSenderId: "",
    appId: ""
});
```

```sh
# Install globally (recommended).
npm run serve
```
